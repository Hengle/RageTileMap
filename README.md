	+-+-+
	|2|3|
	+-+-+
	|0|1|
	+-+-+
	index = y * width + x

A **simple** tile map using unity and csharp.

Features:
- Create maps with code or custom editor.
- Flip and rotate tiles.
- Animate tiles.
- Maximum of 126x126 (15876) tiles. (Thanks Obama!?)
